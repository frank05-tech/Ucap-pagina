Licenciatura en Software y Hardware

Una carrera diseñada para prepararte en el mundo del desarrollo de software, programación, redes, electrónica y ensamblado de sistemas. Combinamos el conocimiento práctico con la base teórica para formar expertos tecnológicos.

¿Qué aprenderás?

  • Se enseñarán conceptos básicos sobre circuitos eléctricos
  • Se mostrarán las herramientas necesarias para diseñar y ensamblar equipos eléctricos
  • Se tendrá una variedad de sistemas de programación y se enseñará a usar

¿Qué es la Ingeniería en Sistemas?

Es la disciplina encargada del diseño, desarrollo e implementación de soluciones tecnológicas integrales. Un ingeniero en sistemas domina tanto el software como el hardware, siendo capaz de crear programas, mantener redes, administrar bases de datos y resolver problemas informáticos en múltiples entornos.
